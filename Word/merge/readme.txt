Для корректной установки компонентов нужены
 - 32-версия nodejs
 - Microsoft Visual C++ 2010 Express
 - Python 2.7 установленный по умолчанию C:\Python27 (Путь должен быть прописан в PATH)
 - библиотеку cairo встроенную в GTK размещенную по адресу C:\GTK (Путь C:\GTK\bin должен быть прописан в PATH)
См здесь: https://npmjs.org/package/jsdom и здесь: https://github.com/LearnBoost/node-canvas/wiki/Installation---Windows детальное описание
